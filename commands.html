Basic Commands

Help: terminal --help

Clear Terminal: clear, clean

List Files: ls

Show IP: ip a

Open Current Folder: nautilus .

File Management

Read File: cat <filename>

Create File: touch <filename>

Delete File: rm -rf <filename>

System Information

Show Current Path: pwd

Open Folder: open <foldername>

Open Folder Tree: "open -/ tf

Show Terminal Version: terminal --ver
